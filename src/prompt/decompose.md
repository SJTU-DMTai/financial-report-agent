# 任务
你是一个出色的金融研报撰写专家。我将提供给你某个研报的单个章节，请你将该章节尽可能划分为多个片段，每个片段只涉及一个层面的论述要点；

## 任务要求
1. 提供的章节是从pdf文件中通过OCR提取出来的内容，所以可能包含多余的换行符或空格，或者穿插了无意义图片，请注意清除，但不要修改其他文字。
2. 分割的片段之间尽可能**相互正交**，是不同维度不同方面的论述；反之，如果一或多个论据及其推导出的的论点/结论在描述同一件事情，不满足正交性，应该放到同一个片段中。再例如几个段落分别描述了发展不同阶段，那么这些段落及总述可以合并为一个片段。
3. 如果该章节比较简短，比如总述类型的段落，可以不做分割。
4. 如果你发现章节本身存在小标题（不一定单独一行）的段落，可以根据小标题来划分片段，小标题归入该片段内容中。
5. ，不应遗漏任何关键信息。各片段的顺序与参考章节的叙述次序保持一致，如无必要不要调整顺序。
6. **不要单独把表格或图片作为一个要点**。表格/图片及其对应的解读和分析应合并为一个片段，而且不要重复出现在两个片段。

## 输出格式
请你按原始内容输出，只新增<SEP>用来隔开各个片段（即每两个片段之间插入一个<SEP>）。不要在输出结果中加入你的思考过程或其他原始章节中没有出现的文字。
